# 📦 BitBoks - Local Chat & AI Assistant

BitBoks adalah solusi komunikasi modern yang dirancang untuk bekerja di jaringan lokal (WLAN) tanpa ketergantungan pada koneksi internet. Proyek ini menggabungkan kecepatan transfer data lokal dengan kecerdasan AI global melalui sistem **Pairing Assistant**.

## ✨ Fitur Utama

- **Room Layaknya Discord**: Buat dan bergabung ke chatroom secara instan di jaringan lokal.
- **End-to-End Encryption**: Keamanan pesan terjamin dengan enkripsi real-time.
- **Transfer File Kilat**: Kirim dokumen, gambar, dan teks antar perangkat secepat kabel LAN.
- **AI Agent Integration**: Hubungkan kecerdasan AI ke chatroom lokal menggunakan ponsel sebagai jembatan (Bridge).
- **Sistem Lisensi (Product Key)**: Aktivasi mandiri menggunakan key berbasis waktu (TTL) yang dikelola via Cloudflare Workers & KV.

---

## 🚀 Panduan Instalasi & Penggunaan

### 1. BitBoks Desktop (Windows/Linux)
1. Unduh installer sesuai OS Anda di bagian [Download](#).
2. Masukkan **Product Key** yang valid untuk aktivasi awal (butuh internet sekali saja).
3. Buat/Join Room di jaringan Wi-Fi yang sama dengan rekan Anda.

### 2. BitBoks Assistant (Android)
1. Unduh **BitBoks Assistant APK** ke ponsel Anda.
2. Pastikan ponsel terhubung ke **Data Seluler** (untuk akses AI) dan **Wi-Fi** (untuk koneksi ke Desktop).
3. Pilih menu **Pairing** dan scan QR Code yang muncul di aplikasi Desktop.
4. Gunakan perintah `/ai [pertanyaan]` di chatroom BitBoks Desktop.
